3) For unmounting

3a) Unmount filesystem
Run umount command with disk name or mount point name to unmount currently mounted disk.
$ umount /dev/sdb
OR
$ umount /data

3b) Inactivation
 lvchange -an /dev/vg_name/lv_name 
 OR 
 vgchange -an vg_name

3c) Closing LUKS
sudo cryptsetup luksClose DATOS


3d) Power off
